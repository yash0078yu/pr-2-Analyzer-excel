# pr-2-Analyzer-excel
Overview

This project is based on analyzing sales data using Microsoft Excel. It includes data processing, analysis, and visualization using different Excel features.

Objectives
To analyze sales and profit data
To identify high-value customers
To understand sales distribution by region and product
To create a dashboard for better insights
Dataset

The dataset contains the following columns:

Transaction ID
Customer
Product
Region
Date
Sales
Profit
Timestamp
Features Used
Formulas (SUM, AVERAGE, MAX, MIN)
INDEX and MATCH functions
Conditional Formatting
Pivot Table
Charts (Bar, Line, Pie)
What-If Analysis
Data Analysis Toolpak
Analysis Performed
Total Sales and Total Profit calculation
Average, Maximum, and Minimum Sales
Identification of high-value customers
Sales analysis by region and product using Pivot Table
Discount impact using What-If Analysis
